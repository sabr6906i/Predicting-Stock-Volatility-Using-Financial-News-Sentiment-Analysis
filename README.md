## Dataset Descriptions

### NIFTY 50-13-09-2025-to-13-12-2025.csv

This file contains historical data for the **NIFTY 50 index**, covering the period from **13 September 2025 to 13 December 2025**.

Used for:

* Index movement analysis
* Trend identification
* Time-series analysis

Typical fields include date and price-related information such as open, high, low, and close values.

This dataset contains historical equity quote data for **Adani Green Energy Ltd** for a specific trading date.

* Equity performance evaluation
